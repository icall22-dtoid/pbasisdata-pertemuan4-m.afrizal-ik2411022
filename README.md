## 📝 Ringkasan Tugas
[cite_start]Tugas ini mencakup implementasi struktur kontrol aliran percabangan (`IF-THEN-ELSE` dan `CASE`) dalam pemrograman basis data MySQL[cite: 7, 11]. [cite_start]Fokus utama adalah pembuatan *stored procedure* untuk menyelesaikan logika bisnis, seperti penentuan predikat mahasiswa, status kelulusan, kategori stok, dan perhitungan diskon[cite: 14, 318, 319].
